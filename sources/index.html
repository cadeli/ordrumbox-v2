OrDrumbox : Free online drum machine

ordrumbox is a online drum machine web app that lets you make your own drum beats directly in your browser. It comes with many presets, drum patterns and kits that you can choose from, including traditional accoustic drums, 8-bits kits and vintage-style kits. There are also some fun sounds available for you to experiment with.

Using a drum machine is easy, as it has a powerfull and user-friendly interface that allows you to customize your beats in various ways. You can set up triggers to control when different sounds play with counters, and use loops controls to repeat sections of your tracks. There are also LFO modulator that let you add character to sounds.

There is also more traditional controls, you can adjust the pitch, volume, and placement of each note or track as needed.

This online drum machine is a versatile tool optimised for creativity that lets you create and experiment with different beats, regardless of your musical experience.

Main Features

Presets

35 demo patterns, 8 drum kits, 5 presets for synth sounds

Controls

pitch, volume and panoramic per note or per tracks.

Step sequencer

Add, modify and delete note.

Triggers

counter triggers, repeat controls and LFO modulators for each track or notes.

Bass line Generator

Create automatic bass line on a separate track.

Sound Synth

Customise sounds with two VCO one LFO and one filters.

Import/Export

Save and load patterns as local files. Export sessions as wave file
